## Project Overview
Sunrise Bakery is a fictional small business website created as part of a **Web Design & Development Test Assignment**.  
The goal of this project was to design and build a **fully functional, responsive website** that demonstrates strong fundamentals in HTML, CSS, and JavaScript, along with attention to layout, usability, and clean design.

---

## Objective
To translate given requirements into a polished digital product by creating a small business website with a clear structure, original content, responsive design, and basic interactivity.

---

## Business Concept
**Sunrise Bakery** is a fictional bakery offering freshly baked breads, cakes, pastries, and custom orders.  
The website focuses on warmth, simplicity, and a welcoming user experience that reflects a real local bakery brand.
